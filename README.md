zkSforceSample
==============

terrasky blog用に作ったzkSforceのログインのサンプルです。

##インストール

※事前にcocoapodsがインストールされている想定です。

まずはこのリポジトリのソースコードのクローンを取得して下さい。

    $ git clone git@github.com:setoyama60jp/zkSforceSample.git
    
 そして、zkSforceのインストール
 
    $ pod install
